# -
本教程可以帮你改变网易我的世界的进入游戏画面
准备工作...
1.首先你需要一个MT管理器 需要有Root 
2.你需要有一个网易我的世界的账号
接下来开始教程
现在用MT管理器定位到此路径
/data/data/com.netease.x19/files/games/com.netease/storge/asset/
往下滑
这时候你会看到一个
loginVideoNew.mp4
随后 你需要把要替换的文件准备好 并且重命名为loginVideoNew.mp4
然后你需要把在
/data/data/com.netease.x19/files/games/com.netease/storge/asset/的loginVideoNew.mp4给他删除了
随后用你的
loginVideoNew.mp4去把它复制到
/data/data/com.netease.x19/files/games/com.netease/storge/asset/
就可以替换成功了
